## 🔬 Technical Justification: PPG Intelligence

The ability to measure high-accuracy vitals using only a smartphone camera is powered by **Photoplethysmography (PPG)**.

### 1. The Physics: Hemodynamic Light Absorption

When a fingertip is placed on the lens:

- **Light Source**: The device LED penetrates soft tissue.
- **Hemoglobin Flux**: Oxygenated and deoxygenated hemoglobin absorb light at different rates.
- **Volumetric Change**: Every heartbeat rushes blood into capillaries, increasing light absorption.
- **Peak Analysis**: The system calculates average Red-pixel intensity at 60 FPS to detect systolic peaks.

### 2. SpO2: The Ratio-of-Ratios

Oxygen saturation is derived by comparing the Red and Infrared light absorption spectra across the arterial pulse waveform, providing a clinical-grade estimate of peripheral oxygenation.
